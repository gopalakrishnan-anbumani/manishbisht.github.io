To-Do List - Web Application

Developed a To-Do list web application for the HackerEarth Frontend Internship Interview (Online Round). Used the HTML5 local storage so that the to-do list persists even after the page is refreshed.

  • Two panes - one is to-do pane and the other is completed tasks pane.
  • Color of each uncompleted task is randomly chosen from a set of colors.
  • When a task is completed, we can move it to the completed list(right pane) by drag and drop only.
  • If we wish to delete a task, we can do it by clicking cross button on top right corner of that task.
